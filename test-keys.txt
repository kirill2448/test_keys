Для ресурса https://ipinfo.io/ 

Позитивные кейсы:


Test № 01: Запрос геолокации по IP-адресу

Действие: Отправить запрос к API ipinfo.io для получения геолокационных данных по известному IP-адресу.

Ожидаемый результат: Получить успешный ответ с информацией о городе, регионе, стране и координатах для заданного IP-адреса.

Test № 02: Запрос данных о провайдере по IP-адресу

Действие: Отправить запрос к API ipinfo.io для получения данных о провайдере по известному IP-адресу.

Ожидаемый результат: Получить успешный ответ с информацией о провайдере, включая название и AS-номер.

Test № 03: Запрос информации о собственном IP-адресе

Действие: Отправить запрос к API ipinfo.io без указания IP-адреса (используя текущий IP-адрес устройства).

Ожидаемый результат: Получить успешный ответ с информацией о текущем IP-адресе устройства, включая геолокацию и данные о провайдере.


Негативные кейсы:


Test № 01: Запрос с некорректным форматом IP-адреса

Действие: Отправить запрос к API ipinfo.io с некорректным форматом IP-адреса.

Ожидаемый результат: Получить ошибку или ответ с пустыми данными, указывающими на некорректный формат IP-адреса.

Test № 02: Запрос с недопустимым API-ключом

Действие: Отправить запрос к API ipinfo.io, используя недействительный API-ключ.

Ожидаемый результат: Получить ошибку авторизации или ответ, указывающий на невалидный ключ.

Test № 03: Запрос с отсутствием IP-адреса

Действие: Отправить запрос к API ipinfo.io без указания IP-адреса и без использования текущего IP-адреса.

Ожидаемый результат: Получить ошибку или ответ, указывающий на необходимость указания IP-адреса.

Test № 04: Запрос с недоступностью API ipinfo.io

Действие: Отправить запрос к API ipinfo.io, когда ресурс недоступен.

Ожидаемый результат: Получить ошибку, связанную с недоступностью сервера ipinfo.io.
